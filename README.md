# SimpleTimer
这是一个简单的像素风时间显示与倒计时器，就像下图所展示的那样。主要特色是可以全屏显示内容，这是我在其他软件中没有找到的功能。
右上角杯子图标为返回上一级和退出按钮，全屏后只要按下ESC键便可以退出。
